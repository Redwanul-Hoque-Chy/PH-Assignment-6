Frequently Asked Questions

Whis is the difference between var, let, and const?
var & let value can reassign but const value can not reassign. var is function scope but let & const is block scope.
Whis is the difference between map(), forEach(), and filter()?
Using map() can make new array element from original array. forEach() do not return anything but can execute function. Using filter() can make new array from original array with specific condition.
Explain arrow functions and how they are different from regular functions?
Arrow function have shorter code & introduced in ES6. Arrow fuction do not have own arguments but regular function have own arguments.
How JavaScript Promises work?
Promise work in three state pending, fulfilled, rejected.
How closures work in JavaScript?
Closure allow inner function access to outer function.
